# recipe もやし肉
`#recipe` 
（2人前）
もやし 1袋
ぶたバラ肉200ｇ
もやしは洗ってゆでます
ブタは3〜4㎝ぐらいに切ってもやしをゆでた後の湯でゆでます
肉をゆでるときは沸騰しすぎていより少しぬるい（80度ぐらいの感じ）のほうが柔らかく仕上がります
もやしも肉も私は茹で上げたまま水にはつけませんが そのへんはお好みで

ネギ2㎝
ショウガ1カケ
ニンニク1カケ
それぞれみじん切り
（チューブでも可）
しょうゆ 大さじ3
酒 小さじ1
白ごま 大さじ1
ラー油、ごま油、コシヨウ それぞれ少々
